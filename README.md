# CSS Blogging Website

## Features Implemented
- All required selectors (element, class, ID, grouping, attribute, combinator)
- Box model (margins, padding, borders)
- Flexbox layout for posts/nav
- Pseudo-classes: :hover, :active, :focus, :nth-child
- Responsive with media queries
- Styled contact form
- 5 blog posts on blog.html

Test locally by opening index.html.
